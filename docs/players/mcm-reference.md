# MCM Reference

The MCM has four pages.

## Expressions

Turn individual emotion types on or off and set their basics.

- Toggle each emotion type (pain, combat anger, fatigue, drunk/skooma, eating, cold, trauma, dirty, aroused, swimming, sneaking, no-clothes, random).
- **Eating speed** — how fast the chewing animation plays.
- **Cold method selector** — choose or auto-detect the cold source (Frostfall / Frostbite / SunHelm / Vanilla).

## Settings

Global tuning and input.

- **Sounds** — enable/disable trauma sobbing/breathing audio.
- **Update intervals** — player poll interval and follower poll interval (seconds).
- **Strength sliders** — expression, modifier, and phoneme strength multipliers (0–2×). Scale the intensity of all expressions.
- **Follower system** — enable/disable NPC expressions.
- **Hotkeys** — pause the mod, and register followers on the fly.

## Maintenance

Diagnostics and lifecycle.

- **Status display** — current busy/suspended state and loaded expression counts.
- **Restart** — re-run mod initialization.
- **Reset expression** — clear the current facial state.
- **Uninstall** — clean removal helper.
- **Verbose logging** — extra log/notification output for debugging.
- **Test expression tool** — pick an expression type and phase and play it on demand.
- **Condition overrides** — simulate trauma / cold / dirty / aroused for testing.

## Aroused Modifiers

Per-condition arousal thresholds and caps. Each row sets the arousal level at which a condition applies and how much it caps visible arousal:

- Chilly / cold / freezing
- Pain
- Trauma (minor / major)
- Swim
- Rain
- Headache

!!! tip
    The strength sliders on the **Settings** page are the quickest way to make every expression more subtle or more pronounced without disabling anything.
